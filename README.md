# LTRL 拉丁语阅读器

**Learn to Read Latin (LTRL)** 的中文翻译在线阅读器。

→ **在线访问**: https://MaxLinkerAlpha.github.io/LTRL_reader

---

## 项目简介

本项目是耶鲁大学出版社出版的权威拉丁语教材《Learn to Read Latin》的民间中文翻译版。

### 功能特点

- 📖 多译者对比阅读
- 🔍 术语提示（鼠标悬停查看）
- 🔊 Forvo 发音链接
- 💡 伴侣注释
- 💬 GitHub 评论
