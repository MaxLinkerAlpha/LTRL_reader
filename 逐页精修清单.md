## 访问统计功能方案

### 需求分析
用户希望显示当前浏览网页的人数或累计访问人数。

### 推荐方案 (分阶段)

#### Phase 1: 分析统计 (立即实施)
- **技术**: Cloudflare Web Analytics
- **优点**: 零成本、隐私友好、数据丰富
- **缺点**: 不在页面显示数字
- **操作**: 在 Cloudflare  dashboard 开启

#### Phase 2: 简单计数器 (短期)
- **技术**: 第三方服务 (visitor-badge)
- **优点**: 零配置、页面显示
- **缺点**: 依赖第三方
- **显示**: 页脚显示"累计访问: XXXX"

#### Phase 3: 自建计数 (长期)
- **技术**: Cloudflare Workers + KV
- **优点**: 完全可控、可扩展
- **缺点**: 需要配置
- **功能**: 按页面统计、在线人数

### 不推荐
- **Firebase Realtime**: 对静态阅读站价值不大，实现复杂
